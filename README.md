## **代码构成**
+ xds文件夹下为前端部分，目前基于Vue开发
+ xsdserver文件夹下为后端部分，目前基于Node与go开发

## **运行方式**
1.前端部分
```cmd
    cd dsbaoming
    npm install
    npm run dev
```

2.后端部分
```cmd
    cd xdsserver
    npm install
    node app.js
```
**⚠Note:**

1.请确保已正确安装node环境

2.npm下载速度慢请自行换源
